# Partition Problem Solver

Este projeto implementa três soluções para o problema de particionamento de subconjuntos:

- ✅ **Recursiva pura (DFS)**
- ✅ **Top-down com Memoization (Programação dinâmica com cache)**
- ✅ **Bottom-up com Tabulação (Programação dinâmica iterativa)**

## 🏗️ Como compilar

Use o Makefile incluso:

```bash
make
```

## 🚀 Como executar

Após compilar, execute:

```bash
./partition
```
